# 🏷️ Price Hunter AI – Smart Deal Finder Chrome Extension

## 🚀 Overview
Price Hunter AI is a Chrome extension that compares product prices across Amazon, Flipkart, and Myntra to help users find the best available deal. It fetches product data using SerpAPI, processes intelligent insights via Claude API, and displays optimized recommendations directly in the browser.

---

## ✨ Features
- 🔍 Real-time product price extraction
- 💰 Best deal recommendation
- 🤖 AI-powered analysis using Claude API
- 🌐 Multi-platform support (Amazon, Flipkart, Myntra)
- ⚡ Lightweight and fast performance

---

## 🛠️ Tech Stack
- JavaScript (Chrome Extension)
- Python (Backend Scripts)
- SerpAPI (Product Data Fetching)
- Claude API (AI Processing)
- REST APIs
- JSON Handling

---

## 📦 Installation

1. Clone this repository
2. Go to `chrome://extensions/`
3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select the extension folder

---

## 🔑 API Configuration

This project requires your own API keys.

Create a configuration file and add:

```python
SERP_API_KEY = "YOUR_SERPAPI_KEY"
CLAUDE_API_KEY = "YOUR_CLAUDE_API_KEY"
```

⚠️ API keys are not included for security reasons.
